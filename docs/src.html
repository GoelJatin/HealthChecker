src.app.decrypt()[source]¶

Decrypts the ciphertext parameter received in the request payload using the other parameters, enc_session_key, nonce, tag, and returns the decrypted text as the response.

POST - /Decrypt

Request

{

“enc_session_key”: “”,

“nonce”: “”,

“ciphertext”: “”,

“tag”: “”

}

Response

200

{
“message”: {{ Decrypted Message }}

}

400

  • encoded data is corrupted

    {

    “error”: “Malformed payload”

    }

  • failed to decrypt

    {

    “error”: f”Failed to decrypt the message due to the error: [{error}]”

    }

  • required parameters missing

    {

    “error”: “Tag / Ciphertext / Nonce / Encrypted Session Key missing in the request body”

    }
